Class MetadataBuilder

A builder class for the metadata object.

Example

const metadata = MetadataBuilder.create("Title", "Author", "Description")
 .set("publisher", "Publisher")
 .add("subject", "Subject 1")
 .add("subject", "Subject 2")
 .build();
